# Chess Game

A local **Player vs. Player** chess game developed in **C++**. The project implements the complete set of standard chess rules and provides a simple interface for playing a full game on a single computer.

## Features

* Local Player vs. Player mode
* Complete implementation of the official chess rules
* Move validation
* Check detection
* Checkmate detection
* Pawn promotion
* Castling
* En passant
* Customizable board colors
* Simple and intuitive user interface

## Getting Started

1. Download the project as a **ZIP** file.
2. Extract the ZIP archive to any location on your computer.
3. Open the extracted folder.
4. Start the game by running **`Schach.exe`**.

No installation is required.

## Controls

* **Left-click** a piece to select it.
* **Left-click** a highlighted square to move the selected piece.
* Use the in-game menu to change the appearance of the board and chess pieces.

## Project

This project was created as a C++ application with the goal of implementing a complete and functional chess game. The focus was on correct game logic, rule validation, and a customizable user interface while keeping the gameplay straightforward and easy to use.
